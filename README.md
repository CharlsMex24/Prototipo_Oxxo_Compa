# OXXO Compa — Prototype

> **An AI-powered companion app for OXXO store employees.**  
> Clickable HTML prototype built for the *Innovation and Creative Processes* course (SP2 — Diverge/Ideation Stage).

🔗 **[Live demo](https://charlsmex24.github.io/Prototipo_Oxxo_Compa/)**

---

## The problem

OXXO operates 22,000+ stores in Mexico with 150,000+ employees, mostly young workers facing high turnover, slow onboarding, isolation, and real safety risks on shift.

**Goal:** cut first-year turnover by 30% and lift engagement by giving every store worker an always-on AI partner for training, tasks, wellness, recognition, and safety — in one app.

## The prototype

10 interactive screens covering the full daily flow of a store employee:

1. Login / Splash
2. Home dashboard
3. Compa AI chat
4. Smart tasks
5. Gamified training
6. Wellness check-in
7. Panic / safety
8. Shift swap
9. Recognition
10. Performance

Built mobile-first with OXXO's brand identity. No frameworks — pure HTML/CSS/JS so it loads instantly in any browser.

## Tech stack

- HTML5 / CSS3 / Vanilla JavaScript
- Mobile-first responsive design
- Hosted on GitHub Pages
